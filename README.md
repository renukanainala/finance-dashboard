# 💰 Finance Dashboard

A modern **Finance Dashboard Web Application** built using **React.js** that helps users track income, expenses, and savings with interactive charts and role-based access.

---

## 🚀 Features

* 📊 Dashboard with financial summary (Income, Expense, Savings)
* 📈 Interactive charts using Recharts (Line Chart & Pie Chart)
* 🔐 Role-based access:

  * **Admin** → Can add transactions
  * **Viewer** → Read-only access
* 🧾 Transaction management (Add, View, Filter, Sort)
* 🔍 Search functionality for transactions
* 💡 Dynamic Insights (Highest spending category, savings analysis)
* 💾 Data persistence using **localStorage**
* 📱 Responsive and clean UI

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript
* **State Management:** Context API
* **Charts:** Recharts
* **Styling:** CSS
* **Storage:** localStorage
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone [https://github.com/renukanainala/finance-dashboard.git](https://github.com/renukanainala/finance-dashboard.git)

# Navigate to project folder
cd finance-dashboard

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🎯 Usage

1. Open the application
2. Select your role:

   * Viewer → View dashboard only
   * Admin → Add and manage transactions
3. Add transactions (Admin only)
4. View charts and insights
5. Data will be saved automatically (localStorage)
